Moments from the Games

About This Project

Gaming Hub is a collection of interactive mini-games developed in C++ using the SFML library.

The project was built to explore event-driven programming, real-time graphics rendering, and game logic design.

Each mini-game is self-contained with its own game loop, input handling, and visual feedback, all tied together under one unified launcher.

Features

🎮 Multiple mini-games
🎨 Real-time SFML graphics
⌨️ Keyboard & mouse input
🔄 Individual game loops
🏆 Score tracking
🖥️ Unified menu system

What I Learned

Building and managing multiple game loops in a single project
Handling real-time user input with SFML events
Rendering graphics, text, and shapes using SFML
Structuring a multi-game project cleanly in C++
